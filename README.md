# MVVM Architecture using Kotlin

This MVVM architecture project created using Dagger2, Room, RxJava, Retrofit, Data Binding, LiveData, ViewModel, JSONPlaceholder, MVVM Android.

### Included features
1. Animated Splash
2. Sign In
3. Sign Up
4. Data Listing
